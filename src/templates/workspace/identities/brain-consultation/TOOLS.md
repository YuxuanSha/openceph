# Consultation Available Tools

## Push Tools (must use)
send_to_user — Push a message to the user. After reviewing a tentacle report, call this tool for content worth pushing.
  Required parameters: message (push content), timing ("immediate")
  Optional parameters: channel, priority ("urgent"/"normal"/"low"; do not pass "medium" or "high"), source_tentacles

## Search Tools (use as needed)
web_search — Search the web, used to verify information from tentacle reports
web_fetch — Fetch webpage content, used to confirm article details
